1- Description

Une calculatrice simple faite avec tkinter en python. Elle permet de faire des calculs de base et a un mode sombre.

2- Fonctionnalites

- Addition, soustraction, multiplication, division
- Puissance (x^y)
- Mode sombre / mode clair
- Memoire (MC, MR, MS)
- Preview du resultat en temps reel
- Supprimer le dernier caractere (del)

3- Installation

Pas besoin dinstaller grand chose, tkinter est deja inclus avec python.
Par contre faut avoir python installe sur ton pc.

py calculatrice.py

5 -Ce que jutilise

- tkinter pour linterface graphique
- math pour les calculs
- python 3

6- Comment ca marche

Tu tapes les chiffres, tu choisis loperateur et tu appuies sur =.
Pour la puissance tu fais x^y, ca te met ** dans lexpression et ca calcule.
Le bouton C remet tout a zero, del efface juste le dernier caractere.
Pour le mode sombre cest le bouton en haut a gauche.

7- Memoire

- MS : enregistre le nombre affiche
- MR : rappelle le nombre enregistre
- MC : remet la memoire a zero
