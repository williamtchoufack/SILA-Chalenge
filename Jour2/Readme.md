1- Analyse du dataset Iris

a-Description

Ce projet permet de visualiser et analyser le dataset Iris avec Python. J'ai fait plusieurs graphiques pour explorer les données.

b- Preinstallation des differentes bibliotheques utiliise

Il faut installer les bibliothèques suivantes :
pip install pandas matplotlib seaborn numpy

2- Utilisation

Télécharger le fichier (Iris.csv) et le mettre dans le même dossier que le script, puis lancer :

python main.py

3- Ce que fait le code

- Affiche quelques stats de base (moyenne, médiane, ecart type, quartiles)
- Un histogramme de la longueur des sépales
- Un scatter plot avec une droite de régression
- Une heatmap des corrélations entre les variables
- Une animation qui trace les valeurs de Sepal Length
